323private:
324 /*
325 We can assume that C++ types are 8-byte aligned, and we can't assume that compilers
326 coalesce data members from subclasses. Use bitfields to fill up to next 8-byte
327 aligned size, which is 16 bytes. That way we don't waste memory, and have plenty of room
328 for future flags.
329 Don't use bitfields for the most important flags, as that would generate more code, and
330 access is always inline. Bytes used are:
331 8 vptr + 2 type + 3 bool flags => 3 bytes left, so 24 bits. However, compilers will word-
332 align the quint16s after the bools, so add another unused bool to fill that gap, which
333 leaves us with 16 bits.
334 */
335 bool m_posted = false;
336 bool m_spont = false;
337 bool m_accept = true;
338 bool m_unused = false;
339 quint16 m_reserved : 13;
340 quint16 m_inputEvent : 1;
341 quint16 m_pointerEvent : 1;
342 quint16 m_singlePointEvent : 1;
