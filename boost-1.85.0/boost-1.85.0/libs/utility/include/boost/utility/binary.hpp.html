12/*=============================================================================
13
14 Binary Literal Utility
15 ______________________
16
17
18 The following code works by converting the input bit pattern into a
19 Boost.Preprocessor sequence, then converting groupings of 3 bits each into
20 the corresponding octal digit, and finally concatenating all of the digits
21 together along with a leading zero. This yields a standard octal literal
22 with the desired value as specified in bits.
23
24==============================================================================*/
