24/** Return parsed JSON as a @ref value.
25
26 This function parses an entire string in one
27 step to produce a complete JSON object, returned
28 as a @ref value. If the buffer does not contain a
29 complete serialized JSON, an error occurs. In this
30 case the returned value will be null, using the
31 [default memory resource].
32
33 @par Complexity
34 Linear in `s.size()`.
35
36 @par Exception Safety
37 Strong guarantee.
38 Calls to `memory_resource::allocate` may throw.
39
40 @return A value representing the parsed JSON,
41 or a null if any error occurred.
42
43 @param s The string to parse.
44
45 @param ec Set to the error, if any occurred.
46
47 @param sp The memory resource that the new value and all
48 of its elements will use. If this parameter is omitted,
49 the [default memory resource] is used.
50
51 @param opt The options for the parser. If this parameter
52 is omitted, the parser will accept only standard JSON.
53
54 @see
55 @ref parse_options,
56 @ref stream_parser.
57
58 [default memory resource]: json/allocators/storage_ptr.html#json.allocators.storage_ptr.default_memory_resource
59*/

119/** Return parsed JSON as a @ref value.
120
121 This function reads data from an input stream and parses it to produce a
122 complete JSON entity, returned as a @ref value. If the stream does not
123 contain a complete serialized JSON, or contains extra non-whitespace data,
124 an error occurs. In this case the returned value will be `null`, using the
125 [default memory resource].
126
127 @par Complexity
128 Linear in the size of consumed input.
129
130 @par Exception Safety
131 Basic guarantee.
132 Calls to `memory_resource::allocate` may throw.
133 The stream may throw as described by
134 [`std::ios::exceptions`](https://en.cppreference.com/w/cpp/io/basic_ios/exceptions).
135
136 @return A value representing the parsed JSON,
137 or a `null` if any error occurred.
138
139 @param is The stream to read from.
140
141 @param ec Set to the error, if any occurred.
142
143 @param sp The memory resource that the new value and all of its elements
144 will use. If this parameter is omitted, the [default memory resource]
145 is used.
146
147 @param opt The options for the parser. If this parameter is omitted, the
148 parser will accept only standard JSON.
149
150 @see @ref parse_options, @ref stream_parser, @ref value::operator>>.
151
152 [default memory resource]: json/allocators/storage_ptr.html#json.allocators.storage_ptr.default_memory_resource
153*/
154/** @{ */
155BOOST_JSON_DECL
156value
157parse(
158 std::istream& is,
159 system::error_code& ec,
160 storage_ptr sp = {},
161 parse_options const& opt = {});

172/** Return parsed JSON as a @ref value.
173
174 This function reads data from an input stream and parses it to produce a
175 complete JSON entity, returned as a @ref value. If the stream does not
176 contain a complete serialized JSON, or contains extra non-whitespace data,
177 an exception is thrown.
178
179 @par Complexity
180 Linear in the size of consumed input.
181
182 @par Exception Safety
183 Basic guarantee.
184 Throws `boost::system::system_error` on failed parse.
185 Calls to `memory_resource::allocate` may throw.
186 The stream may throw as described by
187 [`std::ios::exceptions`](https://en.cppreference.com/w/cpp/io/basic_ios/exceptions).
188
189 @return A value representing the parsed JSON upon success.
190
191 @param is The stream to read from.
192
193 @param sp The memory resource that the new value and all of its elements
194 will use. If this parameter is omitted, the [default memory resource]
195 is used.
196
197 @param opt The options for the parser. If this parameter is omitted, the
198 parser will accept only standard JSON.
199
200 @see @ref parse_options, @ref stream_parser, @ref value::operator>>.
201
202 [default memory resource]: json/allocators/storage_ptr.html#json.allocators.storage_ptr.default_memory_resource
203*/
204BOOST_JSON_DECL
205value
206parse(
207 std::istream& is,
208 storage_ptr sp = {},
209 parse_options const& opt = {});
