14/* tag::reference[]
15= `BOOST_HW_SIMD_PPC_*_VERSION`

17Those defines represent Power PC SIMD extensions versions.

24/* tag::reference[]
25= `BOOST_HW_SIMD_PPC_VMX_VERSION`
26
27The https://en.wikipedia.org/wiki/AltiVec#VMX128[VMX] powerpc extension
28version number.

30Version number is: *1.0.0*.
31*/ // end::reference[]

39
40Version number is: *1.1.0*.
41*/ // end::reference[]
42#define BOOST_HW_SIMD_PPC_VSX_VERSION BOOST_VERSION_NUMBER(1, 1, 0)

44/* tag::reference[]
